/*******************************************************************************************/
                                Rockbot for Nintendo DS
/*******************************************************************************************/

INSTALL:
- uncompress the zip file and copy, with all folders to the SD card.

RUNNING:
-use your card system and navigate to /Apps/Rockbot
- open the Rockbot_NDS.nds file

PROBLEMS/TROUBLESHOOTING:
- if you cannot run the file, send us an email to protoman@upperland.net with
information about your card and system. Currently we test it with an old R4
card system and an acekard menu, where it runs without problems.
